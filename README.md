# Monotours24 Website

Modern multi-page starter website for Monotours24 (no domain required yet).

## Pages
- `index.html` — Home
- `tours.html` — Tours
- `about.html` — About
- `contact.html` — Contact + form placeholder
- `impressum.html` — Legal notice
- `datenschutz.html` — Privacy

## Features
- Responsive layout
- Language switcher (DE/RU/EN) via `app.js`
- Ready for GitHub Pages preview

## Note
Form is currently a frontend placeholder (no backend send yet).
